## ⚡ Featured Projects

### 🔋 Phantom Load Detector — ESP32 IoT Energy Monitoring System
> Real-time AC energy monitoring with on-device phantom load detection.

* Built ADC-based sensing pipeline using ZMPT101B (voltage) and SCT013 (current) sensors with hardware-level signal conditioning
* Wrote embedded firmware for sensor sampling, RMS power calculation, and phantom-load detection logic
* Streamed data over **ESP-NOW** to a live dashboard for monitoring, alerts, and historical logging
* Validated at **~85% detection accuracy** through firmware tuning and sensor calibration

`ESP32` `Embedded C` `ADC` `ESP-NOW` `Signal Conditioning`

🔗 [github.com/Ap0128](https://github.com/Ap0128)

---

### 🔐 STM32 Two-Factor Authentication Security Locker
> Firmware-level access control built on bare-metal STM32.

* Reimplemented firmware for a 2FA locker system on **STM32F103C8T6 (Blue Pill)** using Embedded C + STM32 HAL
* Configured GPIO, timers, and peripheral interfaces from the register level
* Debugged initialization sequences, interrupt handling, and firmware state-machine logic for secure access flow

`STM32` `HAL` `GPIO` `Interrupts` `State Machines`

🔗 [github.com/Ap0128](https://github.com/Ap0128)

---

### 🤖 Hospital Assistive Robot — Autonomous Navigation *(Ongoing Capstone)*
> Multi-controller autonomous navigation for a hospital service robot.

* Designed an **11-state Finite State Machine** in Python for navigation and task execution
* Implemented **A\* path planning** on a node-based hospital floor map
* Integrated navigation logic across **Raspberry Pi + ESP32** controllers

`Python` `FSM` `A* Pathfinding` `Raspberry Pi` `ESP32`

🔗 [github.com/Ap0128](https://github.com/Ap0128)
